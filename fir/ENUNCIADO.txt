== NOVA INSTRUÇÃO A ADICIONAR AO COMPILADOR ==

A instrução seguinte permite executar uma função sobre cada elemento de uma
zona de memória indexável por um ponteiro. A função a aplicar deve ter um
único argumento do mesmo tipo dos elementos indexados. Além da função e do
ponteiro para o início da zona de memória, a instrução aceita as expressões
que correspondem aos índices dos extremos do intervalo a processar.

     map <vector-base> : <low> .. <high> -> <function-name>

Os elementos entre < e > denotam partes variáveis da instrução.

Note-se que -> e .. são elementos lexicais atómicos.

Altere o compilador para que suporte esta instrução (especificações Flex e
BYACC, criação de nós, processamento semântico e geração de código).

== EXEMPLO ==

void square(float element) { writeln element*element; }

void f(<float> vector, int size) {
  (* square" aplicado a cada elemento de "vector" *)
  map vector : 0 .. size -1 -> square;
}
